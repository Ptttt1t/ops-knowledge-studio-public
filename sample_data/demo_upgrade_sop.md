# 示例：网元 NE-A 补丁升级操作记录

> 本文件仅用于演示知识抽取流程，不代表真实生产规范。

适用对象：NE-A 型网元。适用版本：V3.1 升级到 V3.1-P2。

执行前必须确认主备状态正常、当前无严重告警，并完成配置备份。若任一条件不满足，应停止升级并提交人工确认。

操作步骤：

1. 在维护窗口开始后再次检查主备状态和告警状态。
2. 导出当前配置并记录备份文件校验值。
3. 上传 V3.1-P2 补丁包并核对补丁包校验值。
4. 先在备用节点安装补丁，确认备用节点恢复正常后再执行主备切换。
5. 在原主节点安装补丁，完成后检查双节点版本。

主要风险是主备切换期间可能出现短时业务抖动；如果备用节点安装后十分钟内未恢复正常，禁止继续执行主节点升级。

回退步骤：停止后续升级，卸载备用节点补丁，恢复升级前配置，并确认主备状态和严重告警恢复到操作前状态。

验证方法：核对双节点版本均为 V3.1-P2，连续观察十五分钟无新增严重告警，并执行一次业务拨测。
